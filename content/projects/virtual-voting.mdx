## Goal

Se trata de una DApp que permita a las personas votar sobre una propuesta (binaria). Cada dirección de Ethereum solo debería tener permitido votar una vez y el voto debe costar 0.01 ETH.

Cuando un usuario abra la página, debería ver el resultado hasta el momento (número de votos positivos vs. número de votos negativos). La actualización en tiempo real es un plus, pero no es necesario.

La aplicación debe consistir únicamente de un frontend. Debe funcionar en Görli y con MetaMask.

![Votación virtual Web 3.0](/virtual-voting.png)

El contrato ya está desplegado en la dirección 0xacfc7725527ba2ee4311574f65e5d76f9f9585e9. Puedes verlo [aquí](https://goerli.etherscan.io/address/0xacfc7725527ba2ee4311574f65e5d76f9f9585e9#code).

## Stack and tools

Los únicos dos requisitos son:

- React para el frontend.
- Web3.js y ethers.js.

Puedes obtener ether de Görli [aquí](https://faucet.paradigm.xyz).
